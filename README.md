# Cloud Data API with Terraform

This project implements a FastAPI REST API for managing and consolidating data, including endpoints to post and retrieve weather records. 

It also includes Terraform configurations to define infrastructure as code, demonstrating resource planning and version-controlled environment setup. 

The project showcases skills in Python, FastAPI, Pydantic, and Terraform, with a focus on data ingestion, data consolidation, and infrastructure automation.
